## 🎯 Problem Statement

Small and Medium Enterprises (SMEs) across India face **critical challenges** in assessing their financial health:

| Challenge | Impact |
|-----------|--------|
| 💼 **Lack of Expertise** | Most SME owners lack deep financial knowledge to interpret complex statements |
| 💰 **High Costs** | Cannot afford expensive financial consultants or enterprise software |
| ⏰ **Time Constraints** | Need quick insights without spending hours on manual analysis |
| ⚠️ **Risk Blindness** | Unable to identify financial risks early |
| 📊 **Missed Opportunities** | Don't know which financial products suit their business |

> **Result:** Poor financial decisions, missed growth opportunities, and increased business failure rates.

## ✨ Core Capabilities

### 🎯 1. Financial Health Scoring
- **0-100 scale** automated scoring
- Real-time risk assessment (Low/Medium/High)
- Industry-specific benchmarking
- Based on profitability ratios, revenue growth, expense control, and cash flow patterns

### 🤖 2. AI-Powered Analysis
- Powered by **Anthropic's Claude API**
- Customized insights specific to company data and industry
- Identifies key financial observations and potential risks
- Highlights growth opportunities with actionable recommendations

### 📊 3. Visual Analytics Dashboard
- Interactive **Chart.js** visualizations
- Revenue vs. expense trends over 12 months
- Month-over-month performance tracking
- Pattern identification without complex spreadsheet analysis

### 💰 4. Cost Optimization Strategies
- Vendor negotiation opportunities
- Process automation recommendations
- Resource optimization tips
- Technology adoption suggestions

### 🏦 5. Financial Product Recommendations
- Business loans (SBI, HDFC, ICICI)
- Working capital facilities
- Credit lines and invoice financing
- Trade finance options tailored to company profile

### 🏭 6. Multi-Industry Support
Tailored analysis for:
- 🏭 Manufacturing
- 💼 Services
- 🛒 Retail
- 💻 Technology
- 🌾 Agriculture
- 🛍️ E-commerce
- 🚚 Logistics

## 💻 Running Locally

### Prerequisites
```bash
- Node.js 18+
- Python 3.11+
- PostgreSQL (optional)
- Git
```

### Backend Setup

```bash
# Clone the repository
git clone https://github.com/Vigneshyadala/financial-health-tool.git
cd financial-health-tool/backend

# Install dependencies
pip install -r requirements.txt

# Create .env file
echo "ANTHROPIC_API_KEY=your_api_key_here" > .env

# Run the server
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

**Backend will be available at:** `http://localhost:8000`

### Frontend Setup

```bash
# Navigate to frontend directory
cd ../frontend

# Install dependencies
npm install

# Create .env file
echo "VITE_API_URL=http://localhost:8000" > .env

# Run development server
npm run dev
```

**Frontend will be available at:** `http://localhost:5173`
